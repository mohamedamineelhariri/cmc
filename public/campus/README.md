# Photos du campus

Déposez ici vos photos de la CMC Béni Mellal-Khénifra. Les pages les utilisent
comme arrière-plans (avec un dégradé de repli si la photo est absente).

Noms de fichiers attendus :

| Fichier | Utilisé sur | Format conseillé |
|---------|-------------|------------------|
| `hero.jpg` | Accueil (hero principal) | paysage, ≥ 1920×1080 |
| `vie-etudiante.jpg` | Page Vie étudiante (hero) | paysage |
| `partenariats.jpg` | Page Partenariats (hero) | paysage |

Optionnel — une photo par pôle (`agriculture.jpg`, `digital-ia.jpg`, …) pourra
être branchée sur les pages de pôle plus tard.

> Les images sont servies telles quelles (`output: 'export'`, `images.unoptimized`).
> Compressez-les (TinyPNG, squoosh) avant de les déposer pour garder le site rapide.
